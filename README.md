# NumberPlateMaker
For model cars, create printing data of Japanese number plate decals.  
![](https://user-images.githubusercontent.com/43490453/50562587-5d977180-0d58-11e9-8530-d94a8c4811dd.jpg)

# Setup
## project initialize
``` $ npm install```

## build typescript
``` $ npm run build```

## debug
``` $ npm run start```  
You can access to localhost:8080.
