# NumberPlateMaker
For model cars, create printing data of Japanese number plate decals.  
![](https://user-images.githubusercontent.com/43490453/50562530-d3e7a400-0d57-11e9-9a76-9ef2090491e1.jpg)

# Setup
## project initialize
``` $ npm install```

## build typescript
``` $ npm run build```

## debug
``` $ npm run start```  
You can access to localhost:8080.
