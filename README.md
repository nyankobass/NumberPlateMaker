# NumberPlateMaker
For model cars, create printing data of Japanese number plate decals.  
![](https://user-images.githubusercontent.com/43490453/50562442-4441f580-0d57-11e9-9783-383bea2a001d.jpg)

# Setup
## project initialize
``` $ npm install```

## build typescript
``` $ npm run build```

## debug
``` $ npm run start```  
You can access to localhost:8080.
