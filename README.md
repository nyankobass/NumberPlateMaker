# NumberPlateMaker
For model cars, create printing data of Japanese number plate decals.  
![](https://user-images.githubusercontent.com/43490453/50562392-e7ded600-0d56-11e9-85b5-acd6fe4d064d.jpg)

# Setup
## project initialize
``` $ npm install```

## build typescript
``` $ npm run build```

## debug
``` $ npm run start```  
You can access to localhost:8080.
